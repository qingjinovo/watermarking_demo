# 数字水印公信平台网站外观与控件介绍

## 整体视觉风格
- **主色调**：深蓝（#0A1A3A）背景，搭配金色（#D4AF37）强调色与司法绿（#2E8B57）成功状态色
- **视觉元素**：采用微光数据流背景、简约区块链节点连线、卡片式分层设计
- **字体**：标题使用无衬线字体（如思源黑体），正文使用系统默认字体确保清晰度

## 交互反馈控件说明

### 1. 按钮状态
- **正常状态**：填充主色，圆角6px
- **悬停状态**：颜色加深，轻微阴影
- **点击状态**：颜色加深，阴影内收
- **禁用状态**：灰色填充，透明度50%

### 2. 表单验证反馈
- **成功**：输入框绿色边框，右侧绿色对勾图标
- **错误**：输入框红色边框，右侧红色感叹号图标，下方红色错误信息
- **正在验证**：输入框蓝色边框，右侧旋转加载图标

### 3. 加载状态
- **页面加载**：顶部进度条（浅蓝色流动条）
- **操作加载**：按钮内文字变为“处理中...”，显示旋转图标
- **数据加载**：卡片区域显示骨架屏（灰色占位块）

### 4. 弹出层组件
- **模态框**：居中显示，背景遮罩，ESC键可关闭
- **下拉菜单**：点击后向下展开，移开收起
- **工具提示**：鼠标悬停时出现，2秒后自动消失
- **通知横幅**：顶部固定位置，3-5秒自动消失，可手动关闭

## 响应式设计要点
- **桌面端（>1024px）**：完整布局，多列展示
- **平板端（768-1024px）**：部分元素单列排列，地图缩小
- **移动端（<768px）**：垂直单列布局，汉堡菜单，大触摸目标

这个外观设计专注于清晰传达平台的**公信力**和**专业性**，同时确保操作流程**直观易懂**，所有控件的设计都服务于“让版权保护变得简单可信”的核心价值主张。
